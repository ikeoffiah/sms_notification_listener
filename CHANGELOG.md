## 0.0.1

* Initial release of the `sms_notification_listener` plugin.

* Implemented `SmsMessage` data model for structured message data.
* Added static methods for checking and requesting SMS permissions at runtime.
* Provided a stream-based API (`onSmsReceived`) for real-time SMS monitoring.
